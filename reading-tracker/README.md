## Reading Rhythm

A personal reading habit companion built with Next.js and Tailwind CSS. Log books, capture sessions, track streaks, and stay aligned with gentle weekly/daily goals. All data is stored locally in your browser for privacy.

## Features

- Add books with target finish dates and notes
- Log reading sessions (pages, minutes, reflections) with automatic progress updates
- Weekly progress ring, streak tracking, and pace insights
- Adjustable weekly minutes and daily pages goals
- Local-first storage powered by `localStorage`

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Commands

- `npm run dev` – start development mode
- `npm run lint` – run ESLint
- `npm run build` – generate the production build
- `npm run start` – serve the production build locally

## Deployment

Deploy quickly on Vercel:

```bash
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-d5308954
```

After deployment, verify the site loads at [https://agentic-d5308954.vercel.app](https://agentic-d5308954.vercel.app).
